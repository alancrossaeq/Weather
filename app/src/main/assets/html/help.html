Weather App Help Guidelines

Map View

The map view can be scrolled and zommed in or out. Clicks on the map will add a marker and the location as a favourite to view forecasts for that location.

Adding a Favourite/Bookmark

Click on a location on the map to add a marker and a favourite.

Favourite List

Once favourites have been added, they can be viewed at the top of the map view.

Location View

Selecting a favourite from the favourite list will bring you the that locations detail page. Here the current weather and five day forecast can be found. This location can be unfavourited via the options menu while on this page.
